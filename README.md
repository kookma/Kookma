# kookma
kookma is a full featured Tiddlywiki 5 app.
This will includes

* An empty (vanilla) TW 5.1.19+ and 
* Shiraz plugin
* Semantic colors
* Timestamp plugin
* Favorites plugin
* Field view plugin
* Customization stylesheets
* Slider plugin


# Under development
It will be availabe around Summer 2019
